ICD 2O1 – Summative Project

Recreate a Retro Video/Computer Game

 

Objective: 

Demonstrate your understanding of computer programming concepts through the recreation of a classic computer/video game.

 

Success Criteria: 

This project will need to incorporate the following elements to meet the criteria for success:

 

Essential Elements

 

  1. Constant and Variable values
  2. Varying data types (integers, floating-point, string, and Boolean values.)
  3. Processing arithmetic operations (addition, subtraction, division, etc.)
  4. Decisions (if, if-else, if-else-if statements)
  5. Decisions with logical operators
  6. Loops
  7. Annotations within the code for code maintenance.
  8. Flowchart or program structure document for planning and project management.
  9. Consistent communication with your teacher.

 

Advanced Elements (depending on the type of project selected)

  1. Simple graphics / animations / motion
  2. Advanced conditionals / logical operators

 

Ideas for game concepts:

 

  1. Select a game to recreate from your research in the Project Inspiration folder (found in the Content section of the course).
    • This could be:
      • a classic text adventure game
      • a classic Atari video game
      • a classic PC game
  2. Go to the Project Blog link (found in the Final Project folder) and post your selection for teacher approval.
  3. Once you have teacher approval, recreate the selected game using your knowledge of the Python programming language.
    • Depending on the game you choose to recreate, you may only need to recreate a smaller portion of the game -- consult with your teacher for the necessary elements to recreate

 

 

Communication with Your Teacher:

 

Throughout this project, you will need to keep consistent communication with your teacher, using the Project Blog discussion forum.

Each project session, you will need to keep a journal/logbook of the progress of your project work.

Your session journal / log should include details about:

  • What was accomplished on the project for the session?  Did you meet all of your intended goals outlined in the previous journal / log entry.
  • What challenges did you encounter?  How did you overcome them?
  • What do you plan to accomplish the next session?
